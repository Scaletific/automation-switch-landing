Content SEO
On-page signals that directly influence ranking and dwell time
0 / 8
✓
Primary keyword appears in the H1 (article title)
The H1 is the article title. The primary keyword should appear in the first 4 words where possible. Only one H1 per page — this is automatic from the Sanity schema.
SEO
✓
Primary keyword appears in the first paragraph
Google gives extra weight to keywords in the opening paragraph. Don't force it — but make sure the opening naturally mentions the topic the article is about.
SEO
✓
H2/H3 subheadings use related and LSI keywords
LSI = Latent Semantic Indexing — related terms and phrases. E.g. if primary keyword is "automation failure", H2s might use "trigger errors", "workflow breakdown", "silent failures".
SEO
✓
At least 2 internal links to related articles
Link using descriptive anchor text — not "click here". Internal links distribute page authority and signal topical relevance to Google. Use the related articles section and in-body links.
SEO
✓
At least 1 outbound link to an authoritative external source
Link to data sources, tools you reference, or relevant documentation. Use rel="nofollow" for commercial links. Signals to Google that the content is well-researched.
SEO
✓
Hero image has descriptive alt text
Alt text describes what's in the image — not the article title. Example: "A flowchart showing three failure points in a multi-step Make.com workflow". Helps with image SEO and accessibility.
SEO
✓
PrecisionReach cross-linked naturally where relevant
Where the article topic connects to cold email outreach, lead research, or AI automation — link to PrecisionReach. Don't force it. The tool CTA in the sidebar is always present regardless.
SEO
✓
Word count matches the category depth expectation
Fundamentals: 800–1,200 words. Tool Guide: 1,000–1,800 words. Deep Dive: 1,500–3,000 words. Word count alone doesn't rank — but thin content rarely competes for competitive keywords.

E-E-A-T Signals
Experience, Expertise, Authoritativeness, Trust — Google's quality framework
0 / 6
✓
Author document exists and is fully filled
The author document must have: full name, bio (2–3 sentences of genuine expertise), avatar photo, role/title, at least one social profile link. This is the foundation of E-E-A-T — currently missing from schema.
New FieldE-E-A-T
✓
Author bio on the article matches their expertise to this topic
The author bio should explain why this person is qualified to write on this specific subject. Vague bios ("passionate about tech") score poorly vs. specific ones ("built 80+ production automations for enterprise clients").
E-E-A-T
✓
Content includes first-hand experience or original data
Google's "Experience" signal (the first E in E-E-A-T) rewards content based on real experience. Include specific examples, original observations, or data from your own automation work — not just general advice.
E-E-A-T
✓
Claims and statistics are sourced or attributed
Any stat, benchmark, or claim should link to its source, or clearly be attributed to your own experience ("in our client work…"). Unsourced statistics weaken Trust score.
E-E-A-T
✓
Author profile page exists at /authors/[slug]
Google uses author pages as an authority signal. The page should list all articles by this author and include the same bio, photo, and social links as the author document. This is a separate page to build.
E-E-A-TRich Snippet
✓
Publish date is honest — not back-dated for SEO
Google penalises artificial date manipulation. Set publishedAt to the real publish date. If updating substantially, update lastModified instead. Don't reset publishedAt on edits.

Rich Snippet Eligibility
Schema that unlocks enhanced listings in Google search results
0 / 5
✓
Article schema eligible: title + image + author + date all set
Google's Article rich result requires all four. When present, your article may appear in the Top Stories carousel and include an image thumbnail in standard search results.
Rich Snippet
✓
FAQ schema eligible: at least 2 FAQ items added
Each question should be phrased as a real user query ("How do I…", "Why does…", "What is…"). Answers should be 2–4 sentences. Google shows these as expandable accordions below your listing — massive click-rate booster.
New FieldRich Snippet
✓
Featured snippet targeted: Key Takeaways box completed
Google frequently pulls the Key Takeaways/TL;DR box as a featured snippet (position zero). Each bullet should directly answer a "What is…" or "How to…" question a user might search for.
New FieldRich Snippet
✓
Breadcrumb schema will auto-generate from category + slug
Breadcrumbs appear in the Google search listing URL path (replaces the full URL with Home › Articles › Category). This is automatic once category and slug are set correctly.
Rich Snippet
✓
Post-publish: all structured data passes Google Rich Results Test
Run the URL through search.google.com/test/rich-results within 24 hours of publishing. Fix any errors before requesting indexing via Search Console.
